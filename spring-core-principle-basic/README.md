## Goal

* 예제 애플리케이션을 직접 코드로 개발하는 과정 속에서 **스프링의 핵심 원리**를 깊이있고 자연스럽게 이해한다.

* Project: `Gradle - Groovy` Project

* Packaging: Jar

* Java: 11

* IDE: IntelliJ

* Dependencies: No selected
